# Evaluating Muon Optimizer
Quarter 1 Project for 2025-2026 Capstone

## Environment Setup
- Python 3
- PyTorch
- torchvision
- matplotlib
- muon (installed via GitHub: `pip install git+https://github.com/KellerJordan/Muon`)

CIFAR-10 dataset is automatically downloaded by the code and stored in `./data`.
